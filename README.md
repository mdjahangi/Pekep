# Gucci MVP - Luxury E-commerce

A modern React-based e-commerce MVP for luxury fashion brand Gucci.

## Features

- 🛍️ Product catalog with filtering
- 🛒 Shopping cart with persistence
- 💳 Checkout process
- 📱 Responsive design
- 🎨 Tailwind CSS styling
- 🔍 Product search
- 📦 Category filtering

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- GitHub Pages

npm install
npm run dev
